Linear Programming

What is Linear Programming?

Linear programming (LP) is a method to achieve the best outcome (such as maximum profit or lowest cost) in a mathematical model whose requirements are represented by linear relationships. It is a special case of convex optimization where both the objective function and the constraints are linear.

General Form:

Minimize (or Maximize): \(c^T x\)
Subject to: \(Ax \leq b\)
\(x \geq 0\)

Where:

  • \(x\) is the vector of variables to be determined
  • \(c\) and \(b\) are vectors of known coefficients
  • \(A\) is a known matrix of coefficients

Example Problem

Minimize: x + y
Subject to:
x + y ≤
x ≥ 0, y ≥ 0
