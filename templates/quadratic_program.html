Quadratic Programming

What is Quadratic Programming?

Quadratic programming (QP) is a special type of mathematical optimization problem. It is the problem of optimizing (minimizing or maximizing) a quadratic function of several variables subject to linear constraints on these variables.

General Form:

Minimize: \(\frac{1}{2}x^T P x + q^T x + r\)
Subject to: \(Ax \leq b\)
\(Gx = h\)

Where:

  • \(x\) is the vector of variables to be determined
  • \(P\) is a symmetric matrix defining the quadratic coefficients
  • \(q\) is a vector defining the linear coefficients
  • \(r\) is a scalar
  • \(A\) and \(b\) define inequality constraints
  • \(G\) and \(h\) define equality constraints

Example Problem

Minimize: (x)² + (y)²
Subject to:
x + y =
x - y ≥ 1
